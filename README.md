# Shell
a simple shell implementation

## shell_in_c  
1. cd ninja  
2. `cmake -G Ninja ..` 
3. after compile, then run `ninja` 
4. your execute will show up in your ninja diretory
5. run `.\shell_in_c.exe`

对于内部内置了三个命令 cd help exit 并且可以使用系统内部保存的程序 比如 ls cat echo都可以使用
